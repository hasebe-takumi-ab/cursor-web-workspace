## 🔄 Step 2: 再利用可能ブロックとして保存

### 2-1. WordPressでの保存手順
```
1. 上記HTMLを「カスタムHTML」ブロックにペースト
2. プレビューでアイコンが正しく表示されることを確認
3. カスタムHTMLブロックを選択（クリック）
4. ブロックツールバーの「⋮」（3点メニュー）をクリック
5. 「再利用ブロックに追加」を選択
6. 名前を入力: 「SNSアイコン（選択式）」
7. 「保存」をクリック
```

### 2-2. 管理画面での確認
```
WordPress管理画面 → 投稿 → 再利用ブロック
└── 「SNSアイコン（選択式）」が保存されていることを確認
```

## 🎯 Step 3: メンバーが使用する手順

### 3-1. テンプレートの呼び出し
```
1. プロフィールページの編集画面を開く
2. 「+」（ブロック追加）をクリック
3. 「再利用可能」タブを選択
4. 「SNSアイコン（選択式）」をクリック
5. アイコンが挿入される
```

### 3-2. カスタマイズ（不要アイコン削除）
```
1. 挿入されたブロックを選択
2. ブロックツールバーの「⋮」→「通常のブロックに変換」をクリック
3. HTMLコードが編集可能になる
4. 不要なSNSアイコンの行を削除
   例: YouTubeが不要な場合
   <!-- YouTube -->
   <a href="https://example.com" target="_blank" class="sns-icon youtube" title="YouTube">
   ...SVGコード...
   </a>
   ↑この部分を削除
```

### 3-3. リンクの設定
```
各アイコンのリンクを変更:
href="https://example.com" → href="https://自分のアカウントURL"

例:
- X: href="https://x.com/kagu_mama"
- Instagram: href="https://www.instagram.com/kaguratomama/"
```

## 🛠️ Step 4: 管理・メンテナンス

### 4-1. テンプレートの更新
```
WordPress管理画面 → 投稿 → 再利用ブロック → 「SNSアイコン（選択式）」
└── 編集でスタイルやアイコンを更新可能
└── ただし、既に使用済みのページには影響しない
```

### 4-2. 新しいSNSアイコンの追加
```
基本テンプレートにコードを追加:
<!-- 新SNS -->
<a href="https://example.com" target="_blank" class="sns-icon 新sns" title="新SNS">
    <!-- 新しいSVGアイコン -->
</a>

CSSに色も追加:
.新sns { background-color: #色コード; }
```
