# LINEフローティングボタン 設定ガイド

## 📋 概要
WordPressサイトに常時表示されるLINE友だち追加ボタンを設置します。

## ✨ 機能
- 画面右下に固定表示（スクロールしても追従）
- パルスアニメーションで注目を集める
- ホバー時に拡大＆色が明るくなる
- レスポンシブ対応（PC/タブレット/スマホ）

## 🛠️ WordPress設定手順（超簡単！）

### これだけでOK！
1. TOPページ（または任意のページ）の編集画面を開く
2. ページの一番下までスクロール
3. 「+」ボタンで新しいブロックを追加
4. 「カスタムHTML」ブロックを選択
5. `line_floating_button_complete.html` の内容を**全部**コピー＆ペースト
6. 「更新」または「公開」をクリック

**これで完了！** CSSもHTMLも1つのファイルに入っているので、追加CSSへの設定は不要です。

## 📱 表示イメージ

### PC表示
- ボタンサイズ: 大きめ
- テキスト: 「友だち追加で最新情報GET!」（2行表示）
- 位置: 右下20px

### タブレット・スマホ表示
- ボタンサイズ: やや小さめ
- テキスト: 小さめフォント
- 位置: 右下15px

### 極小画面（360px以下）
- ボタンサイズ: コンパクト
- アイコン: 24px
- テキスト: さらに小さめ

## 🎨 カスタマイズ

### 位置を変更したい場合
HTMLの `bottom` と `right` の値を変更：
```html
style="... bottom: 20px; right: 20px; ..."
```

### テキストを変更したい場合
HTMLの以下の部分を編集：
```html
<span class="line-button-text" style="white-space: nowrap;">
    友だち追加で<br>最新情報GET!
</span>
```

### 色を変更したい場合
HTMLの `background` の値を変更：
```html
background: linear-gradient(135deg, #00B900, #00D900);
```

## ⚠️ 注意事項
- `z-index: 9999` で最前面に表示されます
- 他の要素と重なる場合は、位置を調整してください
- アニメーションが不要な場合は、CSSの `@keyframes linePulse` を削除してください

## 🎯 効果測定
LINE公式アカウントの管理画面で以下を確認できます：
- 友だち追加数の推移
- どのページから追加されたか
- クリック率
